# iwantools-mini-games

`iwantools-mini-games` 为 iwantools 官网提供轻量小游戏。每款游戏独立构建为版本化 Custom Element ESM adapter，由官网作为同源静态资源按需加载。

首个里程碑只交付经典扫雷。项目当前处于规格设计阶段，远端仓库已创建，尚未开始实现。
