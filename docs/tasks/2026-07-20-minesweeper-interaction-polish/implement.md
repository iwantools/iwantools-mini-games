# 扫雷交互与胜利反馈优化

## 背景

用户实机体验确认 PC 基础流程可用，但胜利结算缺少动画、手机插旗不够顺手，且缺少经典扫雷的快速排除操作。

## 决策

- 核心层新增 `chordCell`，只对已翻开的非零数字格生效；相邻旗数必须与数字完全相等。
- Web adapter 支持左右键同时按下触发 chord，同时允许对已翻开数字格再次主操作触发，覆盖触屏与键盘。
- 手机端使用“翻格 / 插旗”显式双态控件，长按仍可临时插旗。
- 胜利反馈使用自包含 CSS 动画，不新增资源、声音或震动；减少动态效果模式下关闭动画。
- adapter 发布为新固定版本，官网不覆盖 v0.1.0，保留可回退资源。
